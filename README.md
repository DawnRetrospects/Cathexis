# Cathexis
An overdesigned, custom personal portfolio website.

<br />

### Tech Stack
- Backend: Flask 
    
    - `pip install Flask`
    
    - `pip install Frozen-Flask`

    - `pip install flask_flatpages`

- Frontend: Bootstrap

<br />

### Run Instructions
- To run locally: `python run.py`

- To run on Pages as static site: `python freeze.py`
